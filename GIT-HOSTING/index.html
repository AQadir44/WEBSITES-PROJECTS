HOW TO PUSH ON GITHUB

first download git bash at link

  1. Open git bash by the right click on the folder
  2. Git init
  3. Git status
  4. Git commit -m "first commit"
  5. Git config --global user.name "user name"
  6. Git config --global user.email "user email"
  7. Git remote add origin your git new repository link
  8. Git push -u origin master
